Hypertext Markup Language (HTML)

HTML Meaning?

HTML atau Hypertext Markup Language adalah bahasa standar untuk membuat dan merancang halaman web. HTML digunakan untuk menentukan struktur konten di web, seperti teks, gambar, link, dan elemen multimedia lainnya. Setiap elemen dalam HTML diwakili oleh tag yang memberi tahu browser bagaimana cara menampilkan konten tersebut.

HTML Component

  1. Tag dan Elemen

    2. Tag HTML adalah instruksi yang memberitahu browser bagaimana cara menampilkan konten di halaman web. Setiap tag biasanya memiliki pasangan penutupnya dan elemen HTML adalah gabungan dari tag pembuka, konten, dan tag penutup.

  2. Atribut

    3. Atribut adalah informasi tambahan yang dapat ditambahkan ke tag HTML untuk memberikan lebih banyak detail tentang elemen tersebut. Contohnya, atribut 'src' di tag <img> menentukan sumber gambar.

  3. Struktur Dokumen

    4. HTML memiliki struktur dokumen yang harus diikuti untuk memastikan konten ditampilkan dengan benar. Ini termasuk elemen utama seperti <head> dan <body> yang mengatur metadata dan konten utama halaman.

Cascading Style Sheet (CSS)

CSS Meaning?

CSS atau Cascading Style Sheets adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen dalam halaman web. Dengan CSS, Anda dapat mengontrol warna, font, margin, padding, dan banyak aspek visual lainnya dari elemen HTML. Ini memungkinkan pengembang untuk menciptakan tampilan yang menarik dan konsisten di seluruh situs web.

CSS Component

  1. Selektor

    2. Selektor adalah bagian dari CSS yang digunakan untuk memilih elemen HTML yang ingin diberi gaya. Ada berbagai jenis selektor, seperti selektor tag, kelas, dan ID, yang memungkinkan Anda untuk mengaplikasikan gaya pada elemen-elemen tertentu.

  2. Properti dan Nilai

    3. Properti CSS mendefinisikan aspek visual dari elemen yang ingin Anda ubah, seperti warna teks atau ukuran margin. Setiap properti diikuti oleh nilai yang menentukan bagaimana properti tersebut diterapkan.

  3. Box Model

    4. Box Model adalah konsep dasar dalam CSS yang menggambarkan bagaimana elemen HTML ditempatkan dalam kotak yang terdiri dari margin, border, padding, dan konten. Memahami Box Model adalah kunci untuk mengelola tata letak halaman web dengan efektif.

Java Script (JS)

Java Script Meaning?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Ini memungkinkan pengembang untuk menambahkan dinamika seperti animasi, respon terhadap pengguna, dan manipulasi konten halaman tanpa harus memuat ulang seluruh halaman. JavaScript adalah komponen penting dari pengembangan web modern, bersama dengan HTML dan CSS.

Java Script Component

  1. Variabel dan Tipe Data

    2. Variabel adalah cara untuk menyimpan dan memanipulasi data dalam JavaScript. JavaScript mendukung berbagai tipe data, seperti angka, string, objek, dan array, yang dapat digunakan untuk memodelkan informasi di aplikasi Anda.

  2. Fungsi

    3. Fungsi adalah blok kode yang dapat dipanggil berulang kali dengan berbagai set parameter. Fungsi membantu dalam mengatur kode menjadi bagian-bagian yang dapat digunakan kembali dan dikelola dengan baik.

  3. DOM Manipulation

    4. JavaScript dapat digunakan untuk berinteraksi dan memanipulasi Document Object Model (DOM) dari halaman web. Ini memungkinkan Anda untuk menambah, menghapus, atau mengubah elemen HTML secara dinamis sesuai dengan kebutuhan aplikasi Anda.
